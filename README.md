# Matt's Place

Personal Claude Code plugin marketplace.

## Plugins

| Plugin | Description |
|--------|-------------|
| [aela-voice](https://github.com/mattmk7482/aela-voice) | Give Claude a voice and companion personality |

## Add this marketplace

```bash
claude plugin marketplace add github:mattmk7482/matts-marketplace
```

Then browse and install plugins:

```bash
claude plugin add aela-voice@matts-place
```
